DEN FITNESS ERP

Dynamic gym ERP with live database, roles, fees, attendance, and location check-in.

Real-time operations system for DEN Fitness, Saki Naka. Built to run on Supabase with role-based access for owner, trainers, and members.

Owner Control

Manage plans, monthly fees, dues, attendance corrections, and view business overview.

Trainer Flow

View members, add attendance, enter fee payments, but no member record updates.

Member Access

See own attendance, dues, payment history, and mark presence using gym location.
